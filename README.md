# Fire-Alarm-System-Computer-Vision
 AI-powered real-time fire and smoke detection system using YOLOv12 deep learning. Features live RTSP camera monitoring, instant visual/audio alerts, interactive detection charts, and comprehensive database reporting. Built with Python Flask and OpenCV for reliable 24/7 safety monitoring. Developed at LSPU SCC
